## Что это?
Небольшой скрипт, генерируюший матрицу размерности 2*N - 1, заполненную случайными числами, и выводящий значения матицы из центра по спирали.

## Как запустить

```sh
node matrix.js N=(x*)
```
x* - подставьте любое значение > 0

## Пример

```sh
node matrix.js N=5
```
